ASALightStormProfile Class Reference

Inherits from NSObject
Declared in ASALightStormProfile.h

Overview

Класс предназначен для работы с профилями и связанными с профилем объектами (фильтрами, закладками, адБлоками и тд).

Перед тем, как начать работу с ASALightStorm необходимо создать профиль, либо войти в уже существующий.
